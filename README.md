# SQL Practice & Analytics

Welcome to my SQL practice repository! This project serves as a centralized hub for my SQL scripts, database queries, and data analysis exercises. The goal of this repository is to demonstrate my ability to extract, manipulate, and analyze data to solve practical business problems.

---

## 🚀 Technologies Used

* **Concepts Demonstrated:** 
  * Data Aggregation (`GROUP BY`, `HAVING`)
  * Multi-table Joins (`INNER`, `LEFT`, `RIGHT`)
  * Subqueries & Common Table Expressions (CTEs)
  * Window Functions (`ROW_NUMBER()`, `RANK()`, `LEAD/LAG`)
